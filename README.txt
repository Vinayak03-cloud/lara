# Lara Virtual Assistant

## Installation

1. Install Python (if not already installed).

2. Install the required dependencies:
   ```sh
   pip install -r requirements.txt
   ```

## Running the Assistant

Run the following command in your terminal:
   ```sh
   python lara.py
   ```

Ensure you have a working internet connection and a microphone for voice commands.

## Notes

- Replace `"api key"` in `lara.py` with your actual API keys for full functionality.
- If you face any issues with speech recognition, ensure your microphone permissions are enabled.
